# MedhaOS Training Video Scripts

## Overview

This document contains scripts and storyboards for all MedhaOS training videos. Each script includes narration, on-screen actions, visual elements, and timing.

**Production Notes:**
- Target duration: 3-5 minutes per video
- Format: 1080p HD, 16:9 aspect ratio
- Languages: English, Hindi, Tamil, Kannada (with subtitles)
- Accessibility: Closed captions, audio descriptions

---

## Production Guidelines

### Visual Style

**Color Palette:**
- Primary: MedhaOS blue (#1890FF)
- Success: Green (#52C41A)
- Warning: Orange (#FAAD14)
- Error: Red (#FF4D4F)
- Background: White/Light gray

**Typography:**
- Headings: Poppins Bold
- Body: Inter Regular
- Code: JetBrains Mono

**Animations:**
- Smooth transitions (300ms ease)
- Highlight important UI elements
- Use arrows and callouts
- Zoom in on key features

### Audio

**Narrator:**
- Professional, warm, friendly tone
- Clear enunciation
- Moderate pace (140-160 words/minute)
- Pauses for visual comprehension

**Background Music:**
- Subtle, upbeat, non-distracting
- Volume: -20dB below narration
- Fade in/out at start/end

**Sound Effects:**
- UI clicks and taps
- Notification sounds
- Success/error tones
- Keep minimal and realistic

### Accessibility

**Closed Captions:**
- All narration transcribed
- On-screen text included
- Speaker identification
- Sound effect descriptions

**Audio Descriptions:**
- Describe visual elements
- Explain on-screen actions
- Provide context for visuals

### Localization

**Languages:**
- English (primary)
- Hindi
- Tamil
- Kannada

**Subtitles:**
- All languages
- Accurate translation
- Cultural adaptation
- Proper terminology

### File Formats

**Video:**
- Format: MP4 (H.264)
- Resolution: 1920x1080 (1080p)
- Frame rate: 30fps
- Bitrate: 5-8 Mbps

**Audio:**
- Format: AAC
- Sample rate: 48kHz
- Bitrate: 192 kbps
- Channels: Stereo
