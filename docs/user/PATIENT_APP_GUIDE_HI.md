## शुरुआत करना

### ऐप डाउनलोड करें

**Android:**
1. Google Play Store खोलें
2. "MedhaOS Patient" खोजें
3. "Install" पर टैप करें

**iOS:**
1. App Store खोलें
2. "MedhaOS Patient" खोजें
3. "Get" पर टैप करें

### अपना खाता बनाएं

1. MedhaOS ऐप खोलें
2. "Sign Up" पर टैप करें
3. अपना मोबाइल नंबर दर्ज करें
4. OTP से सत्यापित करें
5. अपनी प्रोफ़ाइल पूरी करें:
   - पूरा नाम
   - जन्म तिथि
   - लिंग
   - पसंदीदा भाषा
   - ABHA ID (यदि आपके पास है)

### अपनी ABHA ID लिंक करें

ABHA (आयुष्मान भारत स्वास्थ्य खाता) आपकी अनूठी स्वास्थ्य ID है जो आपके सभी चिकित्सा रिकॉर्ड संग्रहीत करती है।

**यदि आपके पास ABHA ID है:**
1. Profile → Link ABHA ID पर जाएं
2. अपना 14-अंकीय ABHA नंबर दर्ज करें
3. OTP से सत्यापित करें
4. आपके स्वास्थ्य रिकॉर्ड स्वचालित रूप से आयात हो जाएंगे

**यदि आपके पास ABHA ID नहीं है:**
1. Profile → Create ABHA ID पर जाएं
2. स्क्रीन पर दिए गए निर्देशों का पालन करें
3. आधार OTP से सत्यापित करें
4. आपकी ABHA ID तुरंत बन जाएगी

## आवाज-आधारित ट्राइएज

### ट्राइएज क्या है?

ट्राइएज हमें यह समझने में मदद करता है कि आपकी चिकित्सा स्थिति कितनी गंभीर है और आपको सही सुविधा पर सही डॉक्टर से जोड़ता है।

### वॉइस ट्राइएज का उपयोग कैसे करें

1. **माइक्रोफ़ोन बटन पर टैप करें** होम स्क्रीन पर
   - बीच में बड़ा नीला बटन

2. **अपनी भाषा चुनें**
   - हिंदी, अंग्रेजी, तमिल, कन्नड़ और 18+ अन्य भाषाओं में से चुनें
   - आप अपनी मातृभाषा में बोल सकते हैं!

3. **अपने लक्षण बताएं**
   - अपने फोन में स्पष्ट रूप से बोलें
   - हमें बताएं कि आपको क्या परेशानी है
   - उदाहरण: "मुझे सीने में दर्द है"

4. **फॉलो-अप प्रश्नों के उत्तर दें**
   - AI आपकी स्थिति को बेहतर ढंग से समझने के लिए 5-7 प्रश्न पूछेगा
   - आप आवाज से या विकल्पों पर टैप करके उत्तर दे सकते हैं
   - प्रश्नों में शामिल हो सकते हैं:
     - लक्षण कब शुरू हुए?
     - दर्द कितना गंभीर है?
     - कोई अन्य लक्षण?

5. **अपना अर्जेंसी स्कोर देखें**
   - आपको 0-100 का स्कोर मिलेगा
   - **लाल (70-100):** उच्च तात्कालिकता - आपातकालीन विभाग जाएं
   - **पीला (40-69):** मध्यम - OPD अपॉइंटमेंट लें
   - **हरा (0-39):** कम - घरेलू देखभाल या नियमित जांच

6. **सिफारिशें प्राप्त करें**
   - ऐप सुझाव देगा:
     - निकटतम अस्पताल
     - उपयुक्त विशेषज्ञ
     - अनुमानित प्रतीक्षा समय
     - नेविगेशन दिशा-निर्देश

### बेहतर परिणामों के लिए टिप्स

✅ **करें:**
- स्पष्ट और सामान्य गति से बोलें
- सभी लक्षणों का उल्लेख करें
- गंभीरता के बारे में ईमानदार रहें
- प्रासंगिक चिकित्सा इतिहास शामिल करें

❌ **न करें:**
- प्रश्नों में जल्दबाजी करें
- महत्वपूर्ण विवरण छोड़ें
- फॉलो-अप प्रश्नों को अनदेखा करें

## अपॉइंटमेंट बुक करना

### अपॉइंटमेंट बुक करें

1. **ट्राइएज परिणामों से:**
   - ट्राइएज के बाद, "Confirm Appointment" पर टैप करें
   - सिस्टम स्वचालित रूप से सर्वोत्तम सुविधा और डॉक्टर का चयन करता है

2. **मैनुअल बुकिंग:**
   - होम स्क्रीन पर "Book Appointment" पर टैप करें
   - विशेषता चुनें (कार्डियोलॉजी, ऑर्थोपेडिक्स, आदि)
   - अस्पताल चुनें
   - डॉक्टर चुनें
   - तारीख और समय चुनें
   - बुकिंग की पुष्टि करें

### अपने अपॉइंटमेंट देखें

1. "Appointments" टैब पर जाएं
2. आगामी अपॉइंटमेंट देखें:
   - तारीख और समय
   - डॉक्टर का नाम और विशेषता
   - अस्पताल का स्थान
   - कतार की स्थिति
   - अनुमानित प्रतीक्षा समय

### अस्पताल तक नेविगेट करें

1. अपना अपॉइंटमेंट खोलें
2. "Navigate" पर टैप करें
3. अपना पसंदीदा मैप ऐप चुनें (Google Maps, आदि)
4. दिशा-निर्देशों का पालन करें

### पुनर्निर्धारित या रद्द करें

1. अपॉइंटमेंट खोलें
2. "Reschedule" या "Cancel" पर टैप करें
3. नई तारीख/समय चुनें (पुनर्निर्धारण के लिए)
4. परिवर्तनों की पुष्टि करें

## दवाओं का प्रबंधन

### दवाएं देखें

1. "Medications" टैब पर जाएं
2. अपनी सभी वर्तमान दवाएं देखें:
   - दवा का नाम
   - खुराक
   - आवृत्ति
   - लेने का समय
   - शेष खुराक

### रिमाइंडर सेट करें

जब आपका डॉक्टर दवा लिखता है तो रिमाइंडर स्वचालित रूप से सेट हो जाते हैं।

**कस्टमाइज़ करने के लिए:**
1. किसी दवा पर टैप करें
2. "Edit Reminder" पर टैप करें
3. पसंदीदा समय सेट करें
4. नोटिफिकेशन ध्वनि चुनें
5. परिवर्तन सहेजें

### ली गई के रूप में चिह्नित करें

जब आप अपनी दवा लेते हैं:
1. नोटिफिकेशन पर टैप करें
2. "Mark as Taken" पर टैप करें
3. या दवा कार्ड पर दाईं ओर स्वाइप करें

### अपनी दवा सत्यापित करें

यह सत्यापित करने के लिए कैमरे का उपयोग करें कि आप सही दवा ले रहे हैं:

1. "Scan Medicine" पर टैप करें
2. गोली/टैबलेट पर कैमरा पॉइंट करें
3. ऐप इसे पहचान लेगा
4. पुष्टि करें कि यह आपके प्रिस्क्रिप्शन से मेल खाता है
5. बेमेल होने पर अलर्ट प्राप्त करें

## स्वास्थ्य रिकॉर्ड देखना

### अपने रिकॉर्ड एक्सेस करें

1. "Records" टैब पर जाएं
2. अपनी सभी चिकित्सा जानकारी देखें:
   - चिकित्सा इतिहास
   - डायग्नोस्टिक रिपोर्ट
   - प्रिस्क्रिप्शन
   - टीकाकरण रिकॉर्ड
   - एलर्जी

### डायग्नोस्टिक रिपोर्ट देखें

1. "Diagnostic Reports" पर टैप करें
2. एक रिपोर्ट चुनें (X-ray, रक्त परीक्षण, आदि)
3. परिणाम और AI विश्लेषण देखें
4. PDF डाउनलोड या शेयर करें

### डॉक्टर के साथ रिकॉर्ड शेयर करें

1. रिकॉर्ड खोलें
2. "Share" पर टैप करें
3. तरीका चुनें:
   - QR कोड (अस्पताल में दिखाएं)
   - WhatsApp
   - Email
4. क्या शेयर करना है चुनें
5. पुष्टि करें

## रिकवरी योजनाएं

डिस्चार्ज के बाद, आपका डॉक्टर आपके लिए एक रिकवरी प्लान बना सकता है।

### अपनी रिकवरी प्लान देखें

1. "Recovery" टैब पर जाएं
2. अपनी व्यक्तिगत योजना देखें:
   - रिकवरी टाइमलाइन
   - दवा अनुसूची
   - गतिविधि प्रतिबंध
   - फॉलो-अप अपॉइंटमेंट
   - देखने के लिए चेतावनी संकेत

### प्रगति ट्रैक करें

1. दैनिक चेक-इन:
   - सरल प्रश्नों के उत्तर दें
   - अपने दर्द के स्तर को रेट करें
   - लक्षण लॉग करें
2. प्रगति चार्ट देखें
3. प्रोत्साहन संदेश प्राप्त करें

### शैक्षिक वीडियो देखें

1. "Learn More" पर टैप करें
2. इसके बारे में वीडियो देखें:
   - आपकी स्थिति
   - रिकवरी टिप्स
   - आहार सिफारिशें
   - व्यायाम दिशानिर्देश
3. आपकी भाषा में वीडियो उपलब्ध

## आपातकालीन सुविधाएं

### आपातकालीन बटन

लाल "Emergency" बटन हमेशा आपकी होम स्क्रीन पर दिखाई देता है।

**कब उपयोग करें:**
- गंभीर सीने में दर्द
- सांस लेने में कठिनाई
- गंभीर रक्तस्राव
- चेतना की हानि
- गंभीर एलर्जी प्रतिक्रिया

**क्या होता है:**
1. "Emergency" पर टैप करें
2. आपका स्थान साझा किया जाता है
3. निकटतम आपातकालीन सेवाओं को सूचित किया जाता है
4. एम्बुलेंस भेजी जाती है
5. आपके आपातकालीन संपर्कों को सतर्क किया जाता है
6. आपका चिकित्सा इतिहास अस्पताल को भेजा जाता है

## सेटिंग्स और गोपनीयता

### भाषा सेटिंग्स

1. Profile → Settings → Language पर जाएं
2. अपनी पसंदीदा भाषा चुनें
3. ऐप इंटरफ़ेस और आवाज बदल जाएगी

### नोटिफिकेशन सेटिंग्स

1. Settings → Notifications पर जाएं
2. कस्टमाइज़ करें:
   - दवा रिमाइंडर
   - अपॉइंटमेंट रिमाइंडर
   - स्वास्थ्य टिप्स
   - आपातकालीन अलर्ट

### गोपनीयता सेटिंग्स

1. Settings → Privacy पर जाएं
2. नियंत्रण:
   - आपके रिकॉर्ड कौन देख सकता है
   - डेटा साझाकरण प्राथमिकताएं
   - स्थान ट्रैकिंग
   - बायोमेट्रिक प्रमाणीकरण

## अक्सर पूछे जाने वाले प्रश्न

### क्या मेरा डेटा सुरक्षित है?

हां! हम उपयोग करते हैं:
- 256-बिट एन्क्रिप्शन
- सुरक्षित क्लाउड स्टोरेज
- ABDM अनुपालन
- नियमित सुरक्षा ऑडिट

### क्या मैं ऐप को ऑफ़लाइन उपयोग कर सकता हूं?

बुनियादी सुविधाएं ऑफ़लाइन काम करती हैं:
- सहेजे गए रिकॉर्ड देखें
- दवा रिमाइंडर
- आपातकालीन बटन

उन्नत सुविधाओं के लिए इंटरनेट चाहिए:
- वॉइस ट्राइएज
- अपॉइंटमेंट बुक करना
- नई रिपोर्ट देखना

### AI ट्राइएज कितना सटीक है?

हमारी AI की सटीकता 92% है और इसे 500,000+ मामलों पर प्रशिक्षित किया गया है। हालांकि, यह मदद करने का एक उपकरण है, डॉक्टरों को प्रतिस्थापित नहीं करता। हमेशा चिकित्सा सलाह का पालन करें।

### कौन सी भाषाएं समर्थित हैं?

22+ भारतीय भाषाएं जिनमें शामिल हैं:
- हिंदी
- अंग्रेजी
- तमिल
- तेलुगु
- कन्नड़
- मलयालम
- बंगाली
- मराठी
- गुजराती
- और अधिक!

### क्या ऐप मुफ्त है?

ऐप डाउनलोड और उपयोग करने के लिए मुफ्त है। आप केवल इसके लिए भुगतान करते हैं:
- डॉक्टर परामर्श
- डायग्नोस्टिक परीक्षण
- दवाएं
