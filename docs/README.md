## Quick Start Guides

### For Patients

1. Download MedhaOS app from Play Store/App Store
2. Sign up with mobile number
3. Link ABHA ID (optional)
4. Tap microphone and describe symptoms
5. Book appointment with recommended doctor

### For Clinicians

1. Login at https://clinician.medhaos.health
2. Review patient queue
3. Select next patient
4. Enable Ambient Scribe
5. Conduct consultation
6. Review AI-generated notes
7. Prescribe medications with safety checks

### For Nurses

1. Login to tablet
2. Review prioritized task list
3. Scan patient wristband
4. Scan medication barcode
5. Administer and document
6. Complete shift handoff

### For Administrators

1. Login at https://admin.medhaos.health
2. Review capacity overview
3. Check predictive analytics
4. Address critical alerts
5. Generate reports
